# DevTask

**DevTasks** is a web-based project and task management system designed specifically for development teams. It enables teams to efficiently create and manage projects, assign tasks, track progress through task statuses (e.g., *To Do*, *In Progress*, *Done*), and manage deadlines in a collaborative environment.

The platform incorporates **user authentication (JWT)**, **role-based access control (Admin/Member)**, and supports **DevOps deployment pipelines**, embracing modern software development practices.

## Problem Statement
Software development teams often struggle with organizing tasks, tracking progress, and collaborating effectively without a streamlined system. **DevTasks** solves this by offering a centralized platform with features tailored to development workflows, ensuring clarity, transparency, and improved team productivity.
